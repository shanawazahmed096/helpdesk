# Odoo Helpdesk Replica

A production-ready Helpdesk Management System inspired by Odoo.

## Tech Stack

### Frontend

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend

- Node.js
- Express
- TypeScript
- Prisma

### Database

- PostgreSQL

## Features

- Authentication
- Dashboard
- Ticket Management
- Comments
- Attachments
- Notifications
- Search
- Role-Based Access
